# Emby订阅进度同步

读取 MoviePilot 已配置媒体服务器中的电视剧已入库集数，并回写 MoviePilot 订阅表的 `note` 与 `lack_episode`，用于修正“我的订阅”进度。

## 功能

- 支持手动立即运行和 Cron 定时运行。
- 支持选择指定媒体服务器；不选时使用全部已启用媒体服务器。
- 默认只合并媒体库已有集数，不删除订阅原有 `note`。
- 可选“覆盖已完成集数”，让订阅进度完全以媒体库集数为准。
- 可选“完整后移至历史”，当订阅范围内集数全部入库后调用 MoviePilot 完成订阅逻辑。

## 注意

- 插件通过 MoviePilot 的媒体服务器模块查询，不直接维护 Emby API 配置。
- 请先确保 MoviePilot 已正确配置并连接 Emby/Jellyfin/Plex。
- 如果启用“完整后移至历史”，订阅会从“我的订阅”移入历史记录。
